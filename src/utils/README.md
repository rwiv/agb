# Utils 모듈

## 개요
`utils` 모듈은 프로젝트 전반에서 공통적으로 사용되는 범용 유틸리티 함수들을 제공합니다. 특정 비즈니스 로직보다는 파일 시스템 조작, 포맷 파싱 등 기술적인 보조 기능을 담당합니다.

## 주요 구성 요소

### 1. File System (`fs.rs`)
파일 및 디렉터리 조작과 관련된 편의 기능을 제공합니다.
- **`ensure_dir`**: 파일이 저장될 경로의 상위 디렉터리가 존재하지 않을 경우 재귀적으로 생성합니다. 빌드 결과물 배포(`Emitter`) 시 주로 사용됩니다.

### 2. YAML & Frontmatter (`yaml.rs`)
마크다운 파일에서 메타데이터를 추출하는 기능을 제공합니다.
- **`extract_frontmatter`**: 마크다운 본문 상단의 YAML Frontmatter(`---`로 구분된 영역)를 파싱하여 JSON 객체(`serde_json::Value`)와 순수 본문 내용을 분리합니다.

### 3. TOML 변환 (`toml.rs`)
데이터 포맷 간 변환 기능을 제공합니다.
- **`json_to_toml`**: `serde_json::Value` 객체를 TOML 규격에 맞는 `toml::Value`로 변환합니다. Gemini-cli용 커맨드 변환 시 사용됩니다.

## 설계 원칙
- **범용성**: 특정 리소스 타입에 종속되지 않는 범용적인 기능을 담습니다.
- **안정성**: `anyhow`를 통한 명확한 에러 컨텍스트를 제공하고, 철저한 유닛 테스트를 통해 동작을 검증합니다.
