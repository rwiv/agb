# Task 3.2: 전체 워크플로우 엔드투엔드(E2E) 테스트

## 개요
모든 타겟(Gemini, Claude, OpenCode)에 대해 실제 빌드 프로세스가 정상적으로 작동하는지 최종 확인합니다.

## 상세 작업 내용

1.  **전체 테스트 실행**:
    - `cargo test`를 실행하여 모든 유닛 테스트와 통합 테스트 통과 확인.
2.  **수동 검증 (필요 시)**:
    - 샘플 `agb.yaml`을 사용하여 각 타겟별로 직접 빌드해보고 결과물 확인.

## 성공 기준
- 모든 테스트가 `PASSED` 상태가 된다.
- 각 에이전트별로 의도한 파일 규격(Markdown Frontmatter 또는 TOML)이 정확히 생성된다.
