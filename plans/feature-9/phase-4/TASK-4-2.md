# TASK-4-2: 전체 빌드 사이클 및 잔재 삭제 검증

## 목적
빌드 시 이전 결과물이 깨끗하게 삭제(Clean)되고, 복사된 파일들도 이 프로세스에 안전하게 포함되는지 확인합니다.

## 작업 내용
- `Emitter::emit` 호출 전 `Clean` 단계가 복사된 파일들이 위치할 디렉터리도 포함하는지 검증.
- 수동 테스트 또는 테스트 코드를 통해:
    1. 1차 빌드 수행.
    2. 소스에서 추가 파일 삭제.
    3. 2차 빌드 수행.
    4. 결과물 디렉터리에서 삭제된 파일이 사라졌는지 확인.

## 검증 계획
- 테스트 시나리오 수행 후 결과물 디렉터리의 무결성 확인.
