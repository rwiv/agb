# TASK: Regression Testing

## 개요
새로운 기능 추가 및 리팩터링이 기존의 정상적인 빌드 프로세스에 영향을 주지 않았는지 확인합니다.

## 작업 상세

### 1. 전체 회귀 테스트
- `map.yaml`이 없는 일반적인 프로젝트 상황에서 기존의 `agb build` 결과물(메타데이터 오버라이트 등)이 이전과 동일한지 검증합니다.
- 모든 지원 타겟(`gemini-cli`, `claude-code`, `opencode`)에 대해 테스트를 수행합니다.

## 검증 방법
- `tests/` 디렉터리의 모든 기존 테스트를 실행합니다.
